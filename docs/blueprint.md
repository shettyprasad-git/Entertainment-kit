# **App Name**: Entertainment-Kit

## Core Features:

- User Authentication: Secure user registration, login, and profile management using the Aiven MySQL database.
- Movie/TV Show Browsing: Browse movies and TV shows from TMDB with filtering by genre and trending status using TMDB API.
- Movie Details Page: View movie/TV show details, including title, overview, cast, and trailers, fetched from the TMDB API.
- Personalized Recommendations: AI powered content recommendations based on viewing history.
- Watchlist: Ability to save any video in to the user's Watchlist.
- Search Functionality: Search movies and TV shows.

## Style Guidelines:

- Primary color: Dark gray (#141414) to provide a cinematic viewing experience.
- Background color: Almost-black (#0A0A0A), similar to Netflix's background, creates immersion.
- Accent color: Red (#E50914) for key actions like 'Play' buttons and highlights.
- Body and headline font: 'Inter' sans-serif font, for a modern neutral style.
- Simple, white line icons for navigation and controls.
- Full-screen, edge-to-edge video and poster display, focusing on visual content.
- Smooth transitions and fades for navigation and loading content.
- Implement glassmorphism for a modern, translucent UI, especially for menus and cards.
